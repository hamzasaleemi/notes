# PHP Notes

## PHP
Hypertext Preprocessor is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.
* PHP code is executed on server side so client side only have the output not the process involved.

## Composer
It is a dependency manager for PHP that makes it possible to define third-party code packages used by a project that can then be easily installed and updated.

## PHP tags
* `<?php` (PHP opening tag)
* `?>` (PHP closing tag)
* `<?=` (echo tag)

## Comments
* Multi-Line: /\* This is a comment \*/
* Single Line: // This is a comment

## Predefined Variables
* `$GLOBALS` — References all variables available in global scope
* `$_SERVER` — Server and execution environment information
* `$_GET` — Query string variables
* `$_POST` — Form data from HTTP POST requests
* `$_FILES` — HTTP File Upload variables
* `$_REQUEST` — HTTP Request variables
* `$_SESSION` — Session variables
* `$_ENV` — Environment variables
* `$_COOKIE` — HTTP Cookies
* `$php_errormsg` — The previous error message
* `$http_response_header` — HTTP response headers
* `$argc` — The number of arguments passed to script
* `$argv` — Array of arguments passed to script

## Constants
* `define()` or `const`: declaring Constants
