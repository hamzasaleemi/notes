# HTML Notes
Hyper Text Markup Language is a code that is used to structure a web page and its content.

## HTML Tags
The components used to design the structure of the websites are HTML tags

## Inline Elements
Elements that take up only as much width as they need in a page.
* `<a>` Link
* `<img>` Image
* `<span>` Container
* `<em>` *Emphasize text*
* `<i>` *Italic text*
* `<b>` **Bold text**
* `<strong>` **Important text**
* `<mark>` Highlight text (background color yellow)
* `<s>` ~~Strikethrough text~~
* `<sub>` Subscript like (H2O) move 2 in `<sub>` tag to make it short and move its position to bottom.
* `<sup>` Superscript like (x2) move 2 in `<sup>` tag to make it power of x.
* `<small>` Small text
* `<address>` *Address*
* `<cite>` Citation
* `<q>` ''Quotation''
* `<br>` Line Break
* `<time>` Add time in datetime attribute for better search engine results.
* `<code>` `Code format`
* `<audio>` Audio
* `<video>` Video

## Block Elements
Elements that take up the entire width of parent element.
* `<p>` Paragraph
* `<blockquote>` An extended quotation
* `<details>` Additional information
* `<ul><li>` Unordered list
* `<ol><li>` Ordered list
* `<dl><dt><dd>` Definition list
* `<hr>` Horizontal line
* `<h1>` Heading
